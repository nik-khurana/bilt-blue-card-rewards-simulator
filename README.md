# Bilt Blue Status Rewards Calculator

This is a modern, single-page web application to calculate and compare your Bilt Blue Card Rewards using the two available approaches:

1. **Way 1: Housing Rewards:** Base points + Bonus rent points based on your Everyday Spend to Rent ratio.
2. **Way 2: Cash Back Alternative:** Standard points + 4% Bilt Cash on non-bonus everyday spend.

## Tech Stack
- **HTML5**
- **Tailwind CSS** (via CDN for modern, responsive styling)
- **Vue.js 3** (via CDN for dynamic calculations)

## Hosting Instructions

You can host this for free in seconds using either **Vercel** or **Netlify**.

### Deploying on Netlify (Easiest)
1. Go to [Netlify Drop](https://app.netlify.com/drop).
2. Drag and drop the `biltBlueCardCalculator` folder onto the page.
3. Your site will be instantly deployed and live!

### Deploying on Vercel
1. Install Vercel CLI: `npm i -g vercel`
2. Open your terminal in the `biltBlueCardCalculator` folder.
3. Run `vercel`.
4. Follow the prompts, and your app will be online.

*(Alternatively, you can upload this folder to a GitHub repository and link it to Vercel/Netlify for automatic deployments).*
